Enrollment Overview

There are two ways in which users which are (Providers or Billing Service) can be added and managed.

Providers

Access Functionality: Super Admin (2 maximum), Administrator, General

Flow of enrollment:

  • (ACH flow) Enroll Now -> Provider -> ACH - > Org Details -> Set up Admin -> Financial Institution Information -> Upload W9 -> Review and Submit -> Enrollment Submitted
  • (VCP flow) Enroll Now -> Provider -> VCP - > Org Details -> Set up Admin -> W9 -> Review and Submit -> Enrollment Submitted

Billing Service

Access Functionality: Administrator, General

Flow of enrollment:

  • Enroll Now -> Provider -> Billing Service Information -> Set up Admin -> Upload W9 -> Review and Submit -> Enrollment Submitted

Portals for enrolling

CSR:

  • The CSR page is only accessible by Optum Pay team and not by the users.

UPA:

  • The UPA page is accessible to external users.

Common Functionality:

  • While enrollment the source of truth for validating bank account routing numbers is the ABAValidator.
  • All the documents uploaded while enrollment are stored in the ole.provider_doc. This table just stores the internally generated name for the file. The actual document is stored on some other drive.
